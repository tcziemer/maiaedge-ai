Product Overview
The MaiaEdge Integrated Switch provides additional port capacity to the Path Border Controller. 
Scale connectivity without adding complexity. The MaiaEdge Integrated Switch provides an additional 48 ports for expanded port capacity for the Path Border Controller.  Provides a complete turnk key solution for cross-connect capacity. Expanded port density. 
High-density 10/25GbE for cross-connects in colocation facilities. 
The Switch provides 48 port 10GbE/25GbE and 8 port 100GbE switch in a compact 1RU form factor. 
The Integrated Switch connects to a Path Border Controller via 100GbE uplinks. The PBC treats the Integrated Switch as extended port capacity, with all paths and policies managed centrally by the Path Computation Engine (PCE). No additional routing configuration or protocol stacks are required on the Integrated Switch itself.

Ideal For
Meet-me rooms and carrier hotels requiring high-density cross-connect capacity. Colocation facilities offering tenant interconnection services. Network edges where port count exceeds a single PBC deployment.
