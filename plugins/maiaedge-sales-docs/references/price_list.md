## SKU Naming Convention

```
ME-[PRODUCT]-[CAPACITY]-[TERM][MODIFIER]

PRODUCT:
- PBC-PCE = Path Border Controller with Path Computation Engine
- MPP-48 = Maia Path Port Extender (48-port)

CAPACITY:
- 100G = 100 Gbps throughput
- 10G = 10 Gbps throughput
- 1G = 1 Gbps throughput

TERM:
- 12M = 12 months
- 36M = 36 months
- 60M = 60 months
- POC60 = 60-day POC

MODIFIER:
- -HA = Standby/High Availability unit
- -LAB = Lab use only
```

## Pricing Notes

- All PBC prices include Path Computation Engine (PCE) and interface optics
- Multi-year terms provide cost savings vs. annual renewal
- **Standby/HA units are discounted 30% from primary units**
- POC pricing is for 60-day evaluation only
- Lab SKUs are restricted to non-production environments
- **Order Forms show Annual Price, not TCV**
